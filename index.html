What is Nowi?

Nowi is an open-source security project. Nowi originated from an idea related to a browser with decentralized Bluetooth networks, but it's still in development. However, I've developed several projects with Nowi, such as NowiChat: a chat app using Bluetooth networks and hybrid cryptography. And NowiPass: a password manager that uses symmetric cryptography.

My proyects

NowiChat
My project is NowiChat. NowiChat is a chat app that uses Bluetooth and hybrid cryptography. The problem with NowiChat is the limitations of Bluetooth. However, the rest is perfect since the symmetric algorithm used is AES with a 256-bit key size in GCM mode. For asymmetric cryptography, we have RSA with a 2048-bit key size, in ECB mode, and with the PKCS1 padding algorithm.


NowiPass
My project is NowiPass, NowiPass is a password manager that uses symmetric cryptography to encrypt and decrypt passwords, this manager implements a security generator and evaluator.
In addition, NowiPass also offers other security features, such as a master key for all your passwords and a security question to access your session logs.
